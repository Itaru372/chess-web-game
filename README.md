# AI対戦チェスサイト（chess-web-game）

AI対戦チェスサイトのフロントエンド雛形です。  
現時点では、ホーム画面と対局画面のUI（HTML/CSS）が実装されています。

## 構成ファイル

- `index.html` : 画面構造（ホーム / 対局）
- `styles.css` : UIスタイル
- `AI対戦チェスサイト実装計画.md` : 実装方針・要件の詳細

## 使い方

1. このリポジトリをクローンします
2. `index.html` をブラウザで開きます

ローカルサーバーを使う場合は、例えば以下のように起動できます。

```bash
python -m http.server 8000
```

起動後、`http://localhost:8000` にアクセスしてください。

## 現在の実装状況

- セマンティックなHTMLレイアウト
- ホーム画面（対局開始、レーティング表示、成績・履歴表示領域）
- 対局画面（チェス盤領域、取られた駒表示領域、スコア表示領域、操作ボタン）

## 今後の実装予定

- `chess.js` によるゲームロジック
- `chessboard.js` による盤面描画と操作
- `Stockfish.js` によるAI対戦
- `localStorage` を用いたレーティング・履歴の永続化
